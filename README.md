# Netflix Movies and Shows Analysis

Tools Used-
SQL
Excel

Project Objective-
Analyze Netflix dataset using excel and sql

Analysis Performed
in EXCEL- Check Formatting,Remove empty rows and duplicates, convert xls file to csv
in SQL- import csv file to sql then write sql queries such as
Movies vs TV shows distribution
Top countries producing Netflix content
Yearly release trend
Most common ratings

Key Insights
Movies are more common than TV shows.
USA produces the highest Netflix content.
Content increased significantly after 2016
